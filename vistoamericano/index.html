Bem-vindo!

O primeiro passo na aplicação de um visto de não-imigrante dos EUA é completar sua aplicação.

Antes de Você Começar:

  1. Aprenda sobre Tipos de Vistos.
  2. Este site é projetado para ser acessado usando Google Chrome, Safari somente.
  3. Reúna seus documentos.
  4. Revise as instruções e FAQ.

Notas:

Outras pessoas podem ajudar você com sua aplicação de visto. Mesmo que outra pessoa tenha ajudado a completar a aplicação, você deve assinar.

Notificações sobre Fraude e Abuso no Uso do Sistema

Para a administração do site, informações são coletadas para fins estatísticos. O Departamento de Estado utiliza programas de software para criar estatísticas resumidas para fins como avaliar quais informações são de maior ou menor interesse ou identificar desempenho do sistema ou áreas problemáticas.

Para fins de segurança do site e para garantir que este serviço permaneça disponível para todos os usuários, o Departamento de Estado utiliza programas de software para monitorar o tráfego da rede a fim de identificar tentativas não autorizadas de carregar ou alterar informações ou, de outra forma, causar danos.

Informações da Aplicação

Registre seu Application ID em um local seguro. Você precisará dele para recuperar sua aplicação.

Se houver problemas técnicos com o sistema, ou você quiser completar sua aplicação em outro momento, poderá salvar seu progresso e continuar de onde parou.
